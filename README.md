# 🎡 幸运转盘 - 学习积分抽奖

专为小朋友设计的学习积分奖励抽奖系统！

## 功能特点

- 🎡 **转盘抽奖** - 点击中央按钮转动幸运转盘
- 🎁 **积分商城** - 用积分兑换各种奖励
- 📊 **数据本地保存** - 数据保存在浏览器本地
- ⚙️ **可管理** - 可以增加抽奖机会

## 使用方法

1. 打开网页即可使用
2. 点击中央红色按钮开始抽奖
3. 积分可以在商城兑换奖励
4. 点击"增加抽奖机会"可以添加机会

## GitHub Pages 部署

1. 创建 GitHub 仓库
2. 上传以下文件：
   - `index.html`
   - `.nojekyll`
3. 开启 GitHub Pages
4. 访问 `https://你的用户名.github.io/仓库名/`
